- OOP
- blocks, procs, and lambdas
- error handling and exception management
- working with files, YAML, and dates
- integration with external APIs using Faraday and JSON
- testing with RSpec and WebMock
- static code analysis and style enforcement with RuboCop
- building a custom Ruby gem to interact with an external API (Fun Translations)

Throughout the course, I completed hands-on exercises, wrote tests, and maintained personal **Ruby cheatsheets [RU](cheatsheet_ru.md)/[EN](cheatsheet.md)** for future reference.
